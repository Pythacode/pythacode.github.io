PiLocker
π

Ce projet est une aplication qu'on ne peut quitter sans rentrer les n première décimals de pi. A chaque fois qu'on rentre plus de décimal que n, n augmente et devient le nombre de décimal qu'on à rentrer. Ainsi, si l'utilisateur entre les 5 premières décimal, il devras en rentrer au moin 5 la prochaine fois. Je l'ai codé pour apprendre les premières décimal de pi. Il est disponible sur github à cette adresse : https://github.com/Pythacode/PiLocker
