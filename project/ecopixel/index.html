Ecopixel

Feat GreGrenier

Ecopixel est un jeu développé avec pygame qui consiste a faire des recherches sur la navigateur internet du jeu pour obtenir des pousses. Ces pousses peuvent être planter pour sauver la planète ! Les fruits que les arbres produisent peuvent être revendus pour s'acheter des outils ou améliorer sa maison. Il est sous licence Creative Commons Attribution-NonCommercial 4.0 International (CC BY-NC 4.0).
Il est disponible sur github à cette adresse : https://github.com/Pythacode/Ecopixel
