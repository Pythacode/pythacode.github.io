Verif
EDT

Ce projet est un script python qui se connecte à l'ENT occitanie de l'utilisateur et récupère les information d'emploi du temps, puis dans la version au 22 octobre 2025, envoie un message vio l'api de Free mobile 5 minutes avant chaque cour avec les infos du cours. Le script est sous licence Creative Commons Attribution-NonCommercial 4.0 International (CC BY-NC 4.0).
Il est disponible sur github à cette adresse : https://github.com/Pythacode/VerifEDT
Étant un projet qui vise à être modifier pour corespondre à l'usage de chacun, il n'existe pas de release ni d'executable à télécharger (Ni de logo).
